## Missing Features - Priority Rankings

### 🔴 HIGH PRIORITY - Core Reader Experience
1. **Volume key navigation** - Essential for one-handed reading
2. **Background color** - Important for eye comfort (white/gray/black/custom)
3. **Keep screen on** - Critical for long reading sessions
4. **Show page number** - Basic navigation feedback
5. **Fullscreen** - Maximizes screen real estate
6. **Scale type** - Critical for image display preferences

### 🟡 MEDIUM PRIORITY - Enhanced Experience
7. **Tap zones** - Customizable touch areas for navigation
8. **Double tap animation speed** - Accessibility/UX preference
9. **Download location** - User file management preference
10. **Clear cache** - Storage management
11. **Side padding (webtoon)** - Aesthetics for webtoon reading
12. **Show actions on long tap** - Quick actions preference
13. **Auto-update tracking on read** - Tracking automation
14. **Quiet hours** - Notification management
15. **Concurrent downloads limit** - Bandwidth management

### 🟢 LOW PRIORITY - Nice to Have
16. **Flash on page change (E-ink)** - Niche use case
17. **Split wide pages** - Specific manga types
18. **Rotate wide pages to fit** - Specific manga types
19. **App icon customization** - Personalization
20. **Source languages filter** - Browse refinement
21. **Cloud backup** - Alternative to local backups
22. **Storage usage display** - Informational
23. **Category reordering in settings** - Organization
24. **Per-category notification settings** - Granular control
25. **Custom Discord status format** - Personalization

## Recommendations by Category

### 1. Reader Settings - Major Gap
The reader settings are significantly lacking compared to Mihon. The most critical missing features are:
- Background color (currently only has color filter)
- Keep screen on
- Show page number
- Fullscreen toggle
- Scale type (Fit screen, Fit width, Fit height, etc.)
- Volume key navigation

**Recommendation:** Create a comprehensive ReaderSettings sub-screen with all paged/webtoon/e-ink specific options.

### 2. Library Settings - Minor Gaps
- Auto-update on Wi-Fi only (currently global update check)
- Update restrictions (skip categories, etc.)

### 3. Browse Settings - Moderate Gap
- Source language filtering is important for international users
- Extension repository management is becoming standard

### 4. Download Settings - Minor Gaps
- Download location is standard in most apps
- Download-ahead while reading improves UX

### 5. Data & Storage - Minor Gaps
- Clear cache is standard maintenance action
- Storage usage helps users manage space
