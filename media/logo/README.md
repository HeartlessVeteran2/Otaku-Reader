# Otaku Reader Logo

## Version 1 — "オト" (O-to) Monogram

![Otaku Reader V1 Logo](./v1-oto-monogram.png)

### Design Concept
The first official Otaku Reader logo features a bold, interlocking **Katakana "オト"** (O-to) monogram positioned on a stylized cherry blossom (sakura). This design captures the app's Japanese manga roots while maintaining a modern, streetwear-inspired aesthetic.

### Elements

| Element | Description |
|---------|-------------|
| **Katakana** | "オ" (O) + "ト" (To) interlocking in the center |
| **Cherry Blossom** | 7-petal sakura framing the characters from behind |
| **Open Book** | Subtle book page shape in the bottom petal |
| **Colors** | Deep indigo base, electric pink petals, gold/navy monogram |
| **Style** | Japanese streetwear meets modern app icon |

### Color Palette

| Color | Hex | Usage |
|-------|-----|-------|
| Deep Indigo | `#0f172a` | Background |
| Electric Pink | `#ec4899` | Blossom petals |
| Magenta | `#be185d` | Petal shadows |
| Gold | `#fbbf24` | Monogram highlights |
| Navy | `#1e293b` | Character fills |

### Android Assets

Created adaptive icon components:
- `ic_launcher_background.xml` — Deep indigo with subtle blossom glow
- `ic_launcher_foreground.xml` — Vector representation of the logo
- `ic_launcher_monochrome.xml` — Single-tone silhouette for Android 13+ themed icons
- Mipmap folders for all densities (mdpi through xxxhdpi)
- Adaptive icon support for API 26+

### Future Evolution

This is **Version 1**. As the app grows, the logo may evolve:
- Sharper interlocking (more Linkin Park LP-style fusion)
- Alternative colorways for seasonal variants
- Simplified monochrome version for print/small sizes
- Animated version for splash screen
